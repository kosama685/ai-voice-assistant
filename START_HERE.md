## ⚡ Quick Start (2 Minutes)

### 1. Start the Server
```bash
cd c:\laragon\www\voice_assistant_app
.\venv\Scripts\python run.py
```

### 2. Open in Browser
```
http://127.0.0.1:5000/
```

### 3. Create Account
- Click "Sign Up"
- Fill registration form
- Select plan
- Submit

### 4. Login
- Use your credentials
- Access dashboard

**Done!** You're now using Voice Assistant v2.1 🎉
