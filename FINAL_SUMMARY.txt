HOW TO RUN
================================================================================

1. Navigate to project:
   cd c:\laragon\www\voice_assistant_app

2. Activate virtual environment:
   .\venv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt

4. Run application:
   python run.py

5. Open browser:
   http://127.0.0.1:5000
